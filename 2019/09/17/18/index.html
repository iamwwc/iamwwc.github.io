题目描述

给定一个字符串 s，计算得出 s 对应的值。

题目分析

罗马数字计算最关键的是

  1. 通常数字大小从左到右依次递减。
  2. 如果 s[i] < s[i + 1]，说明这两个数字可以组合在一起，且值为 s[i + 1] - s[i]。
  3. 如果 s[i] >= s[i + 1]，则值为 s[i]
